## Install ionic

### Make sure you have [node.js installed](https://nodejs.org/en/download/)
`$ npm install -g ionic`

## Go to the project's folder

## Get ready for emulations and builds

### Update cordova platform, if needed
`$ cordova platform update android`

### Restore state
`$ ionic state restore`

### Generate resources
`$ ionic resources`

## Run server (for browser based testing)
`$ ionic serve`

## Generate a build

### For Android
`$ ionic platform add android`

### Make sure you have JDK and a valid Android SDK
`$ ionic build android`

### For iOS
`$ ionic platform add ios`

### Build it
`$ ionic build ios`
